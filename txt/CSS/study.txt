// ~
与该元素同父元素的所有位于该元素之后的元素
p~ul: 表示和p同级(p和ul拥有相同父元素)的位于p之后的所有ul元素


// >
CSS3特有的选择器，表示所有该元素的直接后代
A>B: 表示A元素的所有子B元素，但只包括一代


// A B
表示该元素的所有后代元素
A B: 表示A元素的所有子B元素，包括孙子及以后代


// .A.B
表示一个标签既有class A又有class B
<a class "A B"></a>


// 优先级
css中class优先级: 如果一个html的element同时符合几个css的class定义，更详细的会有更大优先级，
比如.a.b.c 比.a.b或者.a .b优先级大，.a .b又比.a优先级大，同样复杂情况下，最后引入的胜出，
例如同一个css中靠后的定义或者靠后引入的css文档
